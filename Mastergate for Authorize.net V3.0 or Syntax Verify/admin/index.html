ADMINISTRATION INFORMATION

Search Database:  Allows you to retrieve member information and subsequently modify or delete member accounts from the search results

Manually Add User:   Allows you to add users manually to the databases.  The first three fields (transaction, subscription, and pincode) are not required.  Expire is a numerical value equal to the NUMBER OF DAYS the account should last.  For example a 30 day account you should enter 30 into the expire field, a 60 day account you would enter 60, etc.

Delete Account:  A quick way to delete users based on usernames

Restore Databases:  This should almost NEVER need to be used except in the event of a server crash.  This will rebuild all of the billing and password databases in the event of a major server malfunction.
