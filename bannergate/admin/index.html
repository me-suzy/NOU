ADMINISTRATIVE NOTES

View User Stats     Displays statistics by user in the bannergate system.  Use the screening window if you want to screen out for example accounts with less than 5 clicks or leave blank to display all active users.

View System Stats     Displays overall stats for the system including impressions, clickthroughs, ctr's, etc.  You can reset the system stats from this display page.

Download Emails     Downloads all email addreses from the /cgi-bin/admin/emails file to your browser window.  Save the resulting output as a TEXT file which can then be imported into your bulk email program of choice.  You can also retrieve the emails file by ftp of course.

View Qualifying Accounts     Displays all accounts which have earned the required number of clickthroughs for a free account and have redeemed them for an account already

Redeem Account     This allows you to enter a single username to see if they have qualified for a free membership or not.  If they have it will provide you with a quicklink to your manual password administration page

Startup System     Click this button after you have first installed BANNERGATE to start it up.  Also can be used to reset your system stats.
