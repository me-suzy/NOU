ADMINISTRATIVE FEATURES AND INFORMAION

CLEAN LINK LIST

This button allows you to check the FFAN sites to see if they still exist.  If the script detects a possible dead link you will see a warning message in green claiming the link is possibly missing.  If the script completes SUCESSFULLY you will see a COMPLETION message saying

LIST CHECK FINISHED

If you DO NOT see this message the last link on your list is not connecting because of network problems or the domain no longer exists.  DELETE the LAST link and reclean.

REVIEW LIVE LINKS

This button allows you to see your current list of VALID FFAN sites.  You can also delete them from your list from this button.

RETRY DEAD LINKS

This button allows you to see your current list of DEAD or DELETED FFAN sites.  You can also readd them from your list from this button if they come back online.

ADD SITE TO NETWORK

This form allows you to MANUALLY add new FFAN sites.  You can also add them from the emails you get when people download the FFAN.ZIP file from your index.html page

SUBMIT URL TO FFAN NETWORK

This form submits URL's to EVERY FFAN site that currently exists on your LIVE LINK list.  ALWAYS clean your list before submitting to make sure ALL of your LINKS are OK.
